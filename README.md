# Calculator App

A simple calculator application created using HTML, CSS and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage calculation
- Clear screen button
- Delete last value
- Keyboard support
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```bash
calculator-app/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in browser
