# CodeAlpha Calculator

A simple web-based calculator built with **HTML**, **CSS**, and **JavaScript**.  
Perform all basic arithmetic operations with a clean, responsive interface and keyboard support.

![Calculator Screenshot](screenshot.png) <!-- Replace with your actual screenshot file, if available -->

## Features

- Addition, Subtraction, Multiplication, Division
- Responsive display and calculator layout
- Keyboard and mouse input support
- Real-time result calculation
- Stylish, modern design

## Getting Started

1. **Clone the repository:**
   ```sh
   git clone https://github.com/lohithasaitech/CodeAlpha_Calculator.git
   ```
2. **Open `index.html` in your browser:**  
   Simply double-click `index.html` or open it with your preferred web browser.

## Usage

- Use mouse or keyboard to enter numbers and operators.
- `C` clears the display.
- `⌫` or Backspace deletes the last digit.
- `=` or Enter evaluates the expression.
- Supported keyboard keys:  
  Numbers (0-9), `+`, `-`, `*`, `/`, `.`, `Enter`, `Backspace`, `c`/`C`

## Customization

- To change the styling, edit [`style.css`](style.css).
- To enhance the logic or add features, modify [`script.js`](script.js).
